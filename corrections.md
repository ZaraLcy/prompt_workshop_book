# 字幕勘誤記錄

此檔案用於收集讀者回報的字幕錯誤。請按照下方格式填寫並提交 Pull Request。

---

## 如何回報錯誤

1. 找到有錯誤的字幕段落
2. 點擊「回報錯誤」按鈕
3. 編輯並複製修訂內容
4. 貼到此檔案對應的模組區塊
5. 提交 Pull Request
